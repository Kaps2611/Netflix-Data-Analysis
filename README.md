# Netflix-Data-Analysis
## Project Objective: 
To analyze and visualize Netflix movies and TV shows data using an interactive dashboard to understand content distribution, genres, ratings, country-wise availability, and release trends over time.

## Dataset:
<a href="https://github.com/Kaps2611/Netflix-Data-Analysis/blob/main/netflix_titles.csv">Dataset</a>

## Questions:
- What is the overall distribution between Movies and TV Shows on Netflix?
- Which countries contribute the highest number of Netflix titles?
- How has the number of Movies and TV Shows changed over the years?
- Which ratings (TV-MA, TV-14, TV-PG, etc.) are most common on Netflix?
- What are the top genres available on Netflix based on content count?
- Which genres dominate Movies vs TV Shows?

## Process:
- Data Collection – Collected Netflix movies and TV shows dataset containing title, type, genre, rating, country, and release details.
- Data Cleaning – Handled missing values, standardized genres, ratings, and country names, and corrected date formats.
- Data Transformation – Created calculated fields for content type distribution, release year trends, and genre-wise analysis.
- Exploratory Data Analysis (EDA) – Analyzed content distribution by type, rating, genre, country, and year.
- Dashboard Development – Built an interactive dashboard to visualize content trends, ratings distribution, top genres, and country-wise availability.

## Project Insights:
- Movies dominate the Netflix catalog, accounting for a significantly larger share compared to TV Shows.
- Netflix content is heavily concentrated in a few countries, with the United States contributing the highest number of titles.
- The platform experienced rapid content growth after 2015, reflecting Netflix’s global expansion strategy.
- TV-MA and TV-14 are the most common content ratings, indicating a strong focus on mature and young adult audiences.
- Documentaries and Stand-Up Comedy are among the top genres, highlighting audience demand for non-fiction and comedy content.
- International content forms a substantial portion of the library, emphasizing Netflix’s investment in global storytelling.
- The number of TV Shows has steadily increased in recent years, signaling a strategic shift toward episodic content.
